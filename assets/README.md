# Assets

## Icônes requises

Pour que l'application fonctionne correctement, vous devez ajouter les fichiers d'icône suivants dans ce dossier :

- **icon.png** : Icône pour le system tray (256x256px recommandé)
- **icon.ico** : Icône Windows pour l'exécutable (plusieurs tailles incluant 16x16, 32x32, 48x48, 256x256)

## Comment créer les icônes

1. Créez une icône de base en format PNG (256x256px)
2. Utilisez un outil en ligne comme https://convertico.com/ pour convertir en .ico
3. Assurez-vous que le fichier .ico contient plusieurs résolutions

## Icône temporaire

Pour tester l'application sans icône personnalisée, l'application utilisera l'icône par défaut d'Electron.
